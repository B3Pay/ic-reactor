## Running Examples Locally

```bash
git clone https://github.com/b3hr4d/ic-reactor-v3.git
cd ic-reactor-v3

# Install dependencies
pnpm install

# Build packages first
pnpm build

# Run a specific example
cd examples/query-demo
pnpm dev
```
