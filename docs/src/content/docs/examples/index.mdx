## Running Examples Locally

```bash
git clone https://github.com/b3pay/ic-reactor.git
cd ic-reactor

# Install dependencies
pnpm install

# Build packages first
pnpm build

# Run a specific example
cd examples/query-demo
pnpm dev
```
