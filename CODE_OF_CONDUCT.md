# Contributor Covenant Code of Conduct

This project follows the Contributor Covenant. By participating, you are expected to uphold this code. Please be respectful, considerate, and collaborative.

For the full Code of Conduct, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

If you experience or witness unacceptable behavior, please contact the maintainers at `B3Pay` on GitHub or open an issue.
