# ckBTC Wallet Example

This Vite example shows how to connect to the ckBTC ledger and minter canisters with IC Reactor hooks.

The app uses fixed ckBTC testnet canister IDs and does not generate local canister declarations. It connects to the public Internet Computer network (using testnet canister IDs) and uses the public Internet Identity provider for authentication.

## Scripts

Run the development server:

```sh
pnpm dev
```

Build the example:

```sh
pnpm build
```
